2)🛠️ WHAT HAPPENS WHEN YOU RUN init_db.py

When you run:

Internally, Python does this:

1. sqlite3.connect("database/attendance.db")
2. If file exists → open it
3. If file does NOT exist → create it automatically
4. Execute CREATE TABLE statements
5. Save schema inside the file
